WELCOME TO THE HOMICIDE MEDIA TRACKER. 
1) it is important to first download and initialise the server. the instructions on how to do so are found in the README inside of the server file

2) once the server is set up, you will be able to initialise the app. to do this, open the terminal and navigate to where to app "Homicide Tracker New" is located

3) To initialise the app, please open the terminal and type in command: npm start 
This will start both the server and the client app.
The App should open soon after the command is executed. 
If the App does not open, go to your browser/safari and type in the url: http://localhost:3000

4) Please note, the default password to delete the database is 1234

5) As of the writing of this document, the edit function is not yet working!
